ES6一些新特性

发表于 2020-03-22 更新于 2020-05-21 分类于 es6
本文字数： 6.2k 阅读时长 ≈ 11 分钟
一些ES6中经常用到的东西。let和const，箭头函数，解构赋值，展开运算符，等等

es6是什么

  • ECMAScript 6（以下简称ES6）是JavaScript语言的某一代标准，已经在2015年6月正式发布了。Mozilla公司将在这个标准的基础上，推出JavaScript 2.0。
  • ECMAScript和JavaScript到底是什么关系？
    简单来说：ECMAScript是JavaScript语言的国际标准，JavaScript是ECMAScript的实现。

es6中定义变量

我们以前都是使用 var 关键字来声明变量的，在 ES6 的时候，多了两个关键字 let 和 const，也是用来声明变量的；

总结

  • let 和 const的相同点：
    • 声明的变量不会经历预解析
    • 声明的变量不允许有相同的变量名
    • 声明的变量必须先声明再使用
    • 在{}中声明的时候，只能在{}内能使用
  • let 和 const的区别：
    • let声明的是变量，后期可以更改值，可以先声明不赋值
    • const声明的是常量，后期不可以更这个常量名的值，声明的时候必须赋值

let/const/var 的区别

  1. let 和 const 不允许重复声明变量

  2. let 和 const 声明的变量不会在预解析的时候解析（也就是没有变量提升）

  3. let 和 const 声明的变量会被所有代码块限制作用范围

let 和 const 的区别

  1. let 声明的变量的值可以改变，const 声明的变量的值不可以改变

  2. let 声明的时候可以不赋值，const 声明的时候必须赋值

箭头函数

箭头函数也是一种函数，是对函数表达式的一种简写；

总结

格式：let fun = () =>{}；

  • 箭头函数的使用规则：
    • 形参写在 圆括号里面
    • 当形参只有一个的时候，可以忽略圆括号
    • 当函数体只有一句话的时候，可以忽略花括号
  • 箭头函数的注意点
    • 箭头函数里面没有 this 的指向
    • 箭头函数没有arguments 对象

箭头函数实例

箭头函数是 ES6 里面一个简写函数的语法方式， 只能简写函数表达式，不能简写声明式函数

语法：(函数的行参) => { 函数体内要执行的代码 }

箭头函数的特殊性

箭头函数内部没有 this，箭头函数的 this 是上下文的 this

按照我们以前的 this 指向来判断，谁调用就指向谁，两个都应该指向 obj；
但是 fun 因为是箭头函数，所以 this 不指向 obj，而是指向 fun 的外层，就是 window。

箭头函数内部没有 arguments 这个参数集合

函数的行参只有一个的时候可以不写 () 其余情况必须写

函数体只有一行代码的时候，可以不写 {} ，并且会自动 return

函数的默认值

我们在定义函数的时候，有的时候需要一个默认值出现；就是当我们不传递参数的时候，使用默认值，传递参数了就使用传递的参数

在 ES6 中我们可以直接把默认值写在函数的行参位置

ES6中数组的方法

  • includes(元素)，检查数组中是否存在 指定的元素，存在就返回true，不存在就返回false
  • Array.from(likeArray)，把伪数组数据转成数组
  • Array.of(参数1，参数2，参数3)，根据参数列表来创建新的数组(消除new Array来创建数组的歧义问题)
  • find(callBack)，从指定的数组中查找第一个满足条件的元素，如果找到那么就直接返回该元素，如果没有找到那么返回undefined
  • findIndex(callBack)，从指定的数组中查找第一个满足条件的元素，如果找到那么就直接返回该元素的索引，如果没有找到那么返回-1

ES6中的for..of（结合了for和forEach的优点）

数组的遍历：

  1. for 循环
    • 优点：可以随时终止（break和continue）
    • 缺点：需要知道循环的次数
  2. forEach
    • 优点：不需要知道循环的次数
    • 缺点：不可以随时终止（break和continue）
  3. for of 遍历有迭代器的对象，不可以遍历对象
    • 优点：不需要知道循环的次数，可以随时终止（break和continue）

Tip：for in 用来遍历对象

生成器（Generators）

  • 在函数的名字前面 + 一个 * 表示是一个生成器
  • 生成器的好处就是可以暂停函数的执行
  • yield “任务一” 理解为暂停
  • next()用来调用函数
    • 返回值：value ：yield的值 done状态
    • 如果后面还有任务，那么done为false，如果后面没有任务了，done就为true
  • 注意：生成器中遇到return ，return后面的代码不会在执行

symbol符号类型（很像字符串）

  • 概念：symbol的功能类似于一种标识唯一性的ID
  • symbol的基本使用：
    • 创建： Symbol()
  • 特点：
    • 在整个代码中所有的符号类型中数据都是唯一的
    • 符号类型的键值对是私有变量，不能被枚举出来（遍历）
    • 不可以跟其他的数据类型进行运算
