发表于 2018-03-20 更新于 2020-05-21 分类于 jQuery
本文字数： 16k 阅读时长 ≈ 29 分钟
记录jq的知识点，因为jq只是js的方法封装，所以没什么需要理解的，基本都是方法。包括选择器，操作元素等等。

是什么

　jQuery，是一个前端类库（方法库，工具库），封装了js中各种操作DOM的方法，95%以上都是方法
　所以jQuery没什么难的知识点，就是更方便的使用js而已；

  • 插件 => 为了实现页面某一个单一类功能而封装的代码
    => swiper 滑动类效果(网页 PPT 效果, 跑马灯, 轮播图, …)
  • 类库 => 把我们常用的操作封装成一个一个的方法, 供我们使用
    => jQuery (封装了各种各样的 DOM 操作)
  • 框架 => 一套有自己完整生态系统的东西
    => 所有东西都准备好了, 你只要按照规则调用
    => Vue / React / Angular
