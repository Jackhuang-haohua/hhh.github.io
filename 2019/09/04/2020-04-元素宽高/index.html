获取元素，浏览器，滚动条的宽高

本文字数： 4.2k 阅读时长 ≈ 8 分钟
offset，scroll，client三大系列，以及相关的小案例(进度条，回到顶部，JS居中，放大镜，拖拽)

获取尺寸


获取元素的宽高：

盒子模型：元素的宽高 +内边距（padding） + 边框（border） + 外边距（margin）

  1. clientWidth 获取元素的宽度（width + padding）

  2. clientHeight 获取元素的高度（height + padding）

  3. clientLeft 获取的是左边框

  4. clientTop 获取的是上边框
    　注意：没有办法获取下边框和右边框

  5. offsetWidth 获取元素的宽度（width +padding +border）

  6. offsetHeight 获取元素的高度（height + padding + border）
    元素的偏移量 到参考框 的距离

  7. offsetLeft

  8. offsetTop
    第一种：当元素没有定位的时候，那么获取或者设置元素到浏览器最左边和上边的距离
    第二种情况：当元素有定位的时候,并有定位的父元素，那么偏移量就是 元素的左边（上边）到父元素的最左边（最上边）的距离
    想要获取元素的宽度（宽度 + padding + border）

获取浏览器的宽高

BOM，browser object model（浏览器对象模型），主要用来操作浏览器对象，顶级对象为window

  1. innerWidth：获取可视区域宽度
  2. innerHeight：获可视区域取高度
  3. outWidth:获取整个浏览器的宽度
  4. outHeight:获取整个浏览器的高度

滚动条的属性和方法

  1. scrollY: 滚动条距离顶部的距离
  2. scrollX: 滚动条距离左边的距离
  3. window.onscroll = function(){} 滚动条滚动事件
  4. window.scrollTo(x,y) 设置滚动条距离 顶部的位置
  5. window.scrollBy(x,y) 设置滚动 与当前位置的距离
