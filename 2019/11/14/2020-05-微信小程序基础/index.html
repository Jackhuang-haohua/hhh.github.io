这篇主要是了解微信小程序的由来、开发者工具、常用设置和快捷键

一、了解小程序

是什么

小程序是一种不需要下载安装即可使用的应用，它实现了应用“触手可及”的梦想，用户扫一扫或者搜一下即可打开应用。也体现了“用完即走”的理念，用户不用关心是否安装太多应用的问题。应用将无处不在，随时可用，但又无需安装卸载。

　对于开发者而言，小程序开发门槛相对较低，难度不及APP，能够满足简单的基础应用，适合生活服务类线下商铺以及非刚需低频应用的转换。 对于用户来说，能够节约使用时间成本和手机内存空间；对于开发者来说也能节约开发和推广成本。

二、准备

申请小程序账号

登录微信公众号平台，申请小程序账号

AppID

登录小程序后台，设置 -> 开发设置 -> 记下AppID

安装开发者工具稳定版

登录微信公众号平台，申请小程序账号

三、开发者工具

新建项目

打开开发者工具，新建项目，选项主要有：

  • 【项目名称】如”douban”
  • 【目录】即为自己的项目地址
  • 【AppID】正确填写即可
  • 【开发模式】选择“小程序”即可
  • 【后端服务】选择“不使用云服务”
  • 【语言】选择“javascript”。

界面

　有很多功能，比如：【模拟器】、【编辑器】、【调试器】、【云开发】、【小程序模式】、【编译】、【真机调试】、【切后台】、【清缓存】、【上传】、【版本管理】、【社区】、【详情】。
　以后工作具体用到那个再记下来。

添加编译模式

　小程序每次重新编译时，都会从app.json中第一个pages开始，如果正在写三四级页面，每次点击会很麻烦；
　此时，可以点击上方的添加编译模式，将当前页添加，并携带当前页需要的参数，这样每次保存代码就会从当前页开始编译；

四、文件介绍

刚创建的项目有以下文件

  • pages -> 页面文件夹
    • borad -> 页面/组件
    • board.js
    • board.json
    • board.wxml
    • board.wxss
  • app.js -> 全局函数
  • app.json -> 全局配置(导航栏，路由)
  • app.wxss -> 全局样式
  • project.config.json -> 项目配置
  • sitemap.json
