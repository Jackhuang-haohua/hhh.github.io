①直接添加

添加集合 -> 添加记录 -> 权限设置，就可以使用这个线上数据库了

②导入json文件

第一步：已有mongodb数据导出，

  • mongod --dbpath E:\study\database\mongodb\data\db
  • mongoexport -h 127.0.0.1 -d 数据库 -c 集合名 -o 输出路径/xx.json，把某个表的数据导出为json
  • 注意，导入的json，对象与对象之间不能有逗号；并且只能导入一个个对象
  • 云开发中选择json文件导入
