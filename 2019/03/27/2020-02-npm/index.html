JS包管理工具：npm，yarn，bower等

是什么

　npm、yarn等都是Node Package Manager，即包管理工具；
　在很久以前，程序员分享代码都是直接网站或者社区里展示；后来较热门的技术又有了专门的网站，比如jQuery，BootStrap；随着代码包越来越多，人们要去不同的地方找不同的东西，太麻烦了，于是”Isaac Z. Schlueter”写出了npm工具，把代码包放在统一的仓库，也就能统一管理；

常见的Package Manager即其操作

1、npm

①npm init，使文件夹能用npm进行管理，会生成一个package.json文件：

②npm i 安装依赖

  • 项目依赖：当前项目上线后依然需要
    • npm i 包名 –save
    • npm install 包名 -S
    • npm install 包名@x.x.x -S
    • npm uninstall 包名 –save
    • npm uninstall 包名 -S
  • 开发依赖：开发过程中需要，上线后就不需要了
    • npm install 包名 –save-dev
    • npm install 包名 -D
    • npm uninstall 包名 -D
  • 检查依赖包
    • npm list 列出所有已装包
    • npm outdated 版本对比(安装过得包)
    • npm info 包名 查看当前包概要信息
    • npm view 包名 versions 查看包历史版本列表
  • 关于版本 npm install 包名@x.x.x
    • ^x.x.x 约束主版本，后续找最新
    • ~x.x.x 保持前两位不变，后续找最新
    • ‘*’ 安装最新版本
    • x.x.x 定死了一个版本
  • 选择源，就是换个下载地址
    • npm install nrm -g 安装选择源的工具包
    • nrm ls 查看所有源
    • nrm test 测试所有源
    • nrm use 切换源名
    • npm config set registry https://registry.npm.taobao.org 切换为淘宝源
  • 有时候安装卡顿，可以这么解决
    • ctrl + c -> npm uninstall 包名 -> npm cache verify 清除缓存 -> 换4g网络 -> npm install 包名
    • 换淘宝源
    • 用cnpm

2、Yarn

“Yarn是由Facebook、Google、Exponent 和 Tilde 联合推出了一个新的 JS 包管理工具 ，正如官方文档中写的，Yarn 是为了弥补 npm 的一些缺陷而出现的。”

①安装

  • 首先我们要安装Yarn，建议官网下载，放在本地的一个文件夹里

②初始化一个新项目

  • yarn init

③添加依赖包

  • yarn add [package]
  • yarn add [package]@[version]
  • yarn add [package]@[tag]

④将依赖项添加到不同依赖项类别中；
分别添加到 dependencies,devDependencies、peerDependencies 和 optionalDependencies 类别中：

  • yarn add [package] –save | -S
  • yarn add [package] –dev | -D
  • yarn add [package] –peer
  • yarn add [package] –optional

⑤升级依赖包

  • yarn upgrade [package]
  • yarn upgrade [package]@[version]
  • yarn upgrade [package]@[tag]

⑥移除依赖包

  • yarn remove [package]

⑦安装项目的全部依赖

  • yarn
  • yarn install

⑧安装到全局

  • yarn global add [package] //global的位置测试不能变
  • yarn global remove [package]
