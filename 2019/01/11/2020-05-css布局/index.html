css布局

发表于 2019-01-11 更新于 2020-05-11 分类于 css
本文字数： 4.7k 阅读时长 ≈ 9 分钟
浮动、定位、flex、grid、水平垂直居中、圣杯、双飞翼

一、文档流display

　不添加任何css，自然呈现的布局；按照文档的顺序一个一个显示出来，块元素独占一行，行内元素共享一行

二、浮动布局float

　使元素脱离文档流，相对于父元素，向左或向右浮动，但是会造成高度塌陷的bug
　高度塌陷：当元素设置浮动而父元素的宽高没有设置时，高度会塌缩为0。
　解决方案：父元素加高度，父元素overflow:hidden，空元素clear:both，父元素.clearfix样式

三、定位position

　有static/relative/absolute/fixed几种：

  1. static：默认值，left、top等不起作用
  2. relative：在原来位置上增加left、top
  3. absolute：脱离文档流，元素的位置就是以他父代元素position不为static的元素作为参考，如果他的父代元素都是position:static，则以body作为参考
  4. fixed：相对于屏幕的位置不变，用于小广告，回到顶部等

　题目：父红色方块relative，index=1；子蓝色方块absolute；兄绿色方块relative。此时给蓝色方块设置index=-1，并不能去到下面 (涉及层叠上下文)。先给红色方块index不设置或等于auto，蓝色方块的index=-1才会生效

四、flex(IE10+，移动端)

①指定容器为flex布局：display: flex / inline-flex / -webkit-flex。设置后，子元素的float，clear，vertical-align失效
②基本概念：flex container容器，flex item项目，main axis水平主轴，cross axis垂直的交叉轴
③属性

容器的属性

  1. flex-direction(主轴方向)：row(水平左边起)，row-reverse、column(垂直上边起)，column-reverse
  2. flex-wrap(是否换行)：nowrap，wrap(换行，第一行在上方)，wrap-reverse(换行，第一行在下方)
  3. flex-flow： ||
  4. justify-content(主轴水平方向的对齐)：flex-start / flex-end / center / space-between(两端) / space-around(每个项目两侧间隔相等)
  5. align-items(交叉轴垂直方向的对齐)：flex-start / flex-end / center / baseline(以item第一行文字为基线)000000 / stretch(默认值，如果项目未设置高度或为auto，item将占满整个容器的高度)
  6. align-content(多根轴线对齐)： flex-start / flex-end / center / space-between / space-around / stretch

项目的属性

  1. order(项目的顺序)：num 数值越小排序越靠前，默认为0
  2. flex-grow(放大比例)：num 默认为0；如果一个项目未2，其他项目为1，则前者占据的空间比其他多一倍
  3. flex-shrink(缩小比例)：num 默认为1；如果一个项目为0，其他项目都为1，则空间不足时，前者不缩小
  4. flex-basis(基础占据空间)：num || auto 默认auto；定义了在分配多余空间之前，项目占据的主轴空间（main size），它可以设为跟width或height属性一样的值（比如350px），则项目将占据固定空间。
  5. flex(前面三个的缩写)：auto
  6. align-self(单个项目的特殊对齐方式)：auto / flex-start / flex-end / center / baseline / stretch

④应用

  • 使元素水平垂直居中居中：display:flex ；justify-content:center ；align-item:center

五、grid网格布局

①概念：父元素为“容器”，子元素为“项目”，不对孙元素生效
②属性

容器的属性

  • display：grid / inline-grid 可以指定容器为行内元素

划分行和列：

  • grid-template-columns：100px 100px 10px
  • grid-template-columns：33.33% 33.33% 33.33%
  • grid-template-columns：repeat(3，33.33%)
  • grid-template-columns：repeat(2,100px 20px 80px)=>6列，100 20 80重复两次
  • grid-template-columns：repeat(auto-fill，100px)=>每列宽100px，填充满容器
  • grid-template-columns：1fr 2fr =>填充满容器，第二列宽度是第一列两倍
  • grid-template-columns：1fr 1fr minmax(100px，1fr)
  • grid-template-columns：[c1] 100px [c2] 100px [c3] auto [c4]=>给网格线命名
  • row-gap：20px 行间距
  • grid-auto-flow：column ，项目按照从上到下，从左到右的方式排列

行缩进

  • grid-auto-flow：row ，所有元素按顺序排，地方太小就空着，不会占满；
  • grid-auto-flow：row dense ，会打乱原来的顺序，把后面的元素提到前面来，直到占满空间；

双飞翼布局

　圣杯布局和双飞翼布局解决问题的方案在前一半是相同的，也就是三栏全部float浮动，但左右两栏加上负margin让其跟中间栏div并排，以形成三栏布局。
　不同在于解决”中间栏div内容不被遮挡“问题的思路不一样：圣杯布局，为了中间div内容不被遮挡，将中间div设置了左右padding-left和padding-right后，将左右两个div用相对布局position: relative并分别配合right和left属性，以便左右两栏div移动后不遮挡中间div。
　双飞翼布局，为了中间div内容不被遮挡，直接在中间div内部创建子div用于放置内容，在该子div里用margin-left和margin-right为左右两栏div留出位置。
