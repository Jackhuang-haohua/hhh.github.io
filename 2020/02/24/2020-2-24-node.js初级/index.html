node.js基础复习笔记

一、什么是node.js

个人的理解是：以前JavaScript只能写页面，有了node后能JS能做更多的事，比如操作数据库，安装拓展，写后端等等；

js在引入html页面后，就可以在浏览器运行，但不能单独运行；当我们安装node.js环境后，我们的电脑也有了’js解析器’，在终端就能直接运行js代码。

node的优点：

  • 标准的异步编程
  • 合理分配资源
  • 高并发 I/O模型

二、安装node.js

  • node官网下载：nodejs.org/en/
  • 下载LTS(稳定版)即可，安装完成后没有图标显示，因为node是系统的一个环境；
  • 检测：win+r 打开命令行，输入node --version，出现版本号表示安装完成；
  • 卸载：控制面板->鼠标右键卸载

三、node环境下运行js代码

方式一：

  • 打开命令行(终端)，输入 node+回车 进入代码编辑模式
  • 输入js代码就可以运行了，相当于在浏览器打开
  • 按 ctrl+c 两次退出模式

方式二：

  • 打开命令行(终端)，进入到包含.js文件的文件夹， node xxx.js 就能运行

四、前端常用的Linux指令

因为node环境下，我们需要在命令行操作，所以得了解下linux指令；

指令 描述 例子
盘符: 切换到某个盘 f:
cd 文件夹名称 进入子级文件夹 cd test
cd .. 返回上一级目录 cs ..
dir 查看当前目录所有内容 dir
md 要创建的文件夹名称 创建文件夹 md test
rd /s/q 要删除的文件夹名称 删除文件夹 rd /s/q test
xcopy 要复制的文件夹 复制后的文件夹名称 复制 xcopy test test2
type nul>要创建的文件名 创建文件 type nul>test.txt
echo 要写入的内容>指定的文件 写入内容，完全覆盖 echo hello>test.txt
echo 要写入的内容>>指定的文件 追加内容 echo world>>test.txt
type 要查看的文件 查看文件内容 type test.txt
ren 要修改的文件 新名称 修改文件名称 ren test.txt test2.txt
del 要删除的文件 删除文件 f:
copy 要复制的文件 复制后的文件名 复制文件 copy test.txt test2.txt
move 要移动的文件 移动到哪一个目录 移动文件 move test.txt e:/study
cls 清除当前命令行所有内容 cls
ipconfig 查看电脑IP信息 ipconfig
ping 网址 测试网速 ping www.baidu.com
systeminfo 查看电脑配置信息 systeminfo
