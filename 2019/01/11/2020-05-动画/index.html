    　因为别人都写的这么详细了，相同的知识我也不好意思在重复一遍…所以这篇文章打算简单记一下知识点，重点是后面做的一些动画实例，来加深理解；

transition

　给元素变化设置一个过渡时间，只有加了transition才会出现动画，不然只会从一个状态闪现到另一个状态；
有4个属性：

  • transition-property: 过渡属性(默认值为all)
  • transition-duration: 过渡持续时间(默认值为0s)
  • transiton-timing-function: 过渡函数(默认值为ease函数)
  • transition-delay: 过渡延迟时间(默认值为0s)

　基本上都会合起来写，比如：transition: width 2s ease 0.5s ;
　实际上就算合起来也很少会写四个属性，一般只会用到:transition: 2s ;

transform(2d)

原点：
　首先要指定变形原点transform-origin，默认在元素正中心，
　一般会才会复合的写法指定：transform-origin: left(0%) top(0%)

类型

  • translate 位移
  • scale 缩放
  • skew 倾斜
  • rotate 旋转

transform(3d)

　比2d多了一个Z轴和透视属性；

  • xy轴不变，z轴垂直于屏幕
  • perspective-origin，类似于之前的变形原点，默认在中间
    • x轴: left | right | center | |
    • y轴: top | bottom | center | |

　属性只有三个：

  • translate 位移
  • scale 缩放
  • rotate 旋转
