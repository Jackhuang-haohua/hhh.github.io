按照redux的逻辑：

  • -> 点击组件里的按钮
  • -> 调用对应仓库的action中的方法
  • -> 通过dispatch(派发)，把action标识符、参数给到reducer文件夹下的notifications.js
  • -> 根据标识符选择执行哪个逻辑，生成新数据(一定要和之前的不同地址)
  • -> 把新数据派发给组件
